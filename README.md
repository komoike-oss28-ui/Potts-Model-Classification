# Potts Model Classification

このプロジェクトは、Potts模型のシミュレーションデータを用いて、機械学習（CNN等）による相転移の分類や相関関数の解析を行うためのコード群です。

## 実行手順（ワークフロー）

解析は以下の順番で行ってください。

### 1. データ生成
まずはシミュレーションを実行し、学習用のデータを作成します。
温度等やq(回転の方向数)を自分で決定可能です
```bash
python generate_Potts-Model_data.py
```

### 2. データ変換
3種類の相関関数でPotts-Modelを変換可能です。また周期的境界条件 or 開放境界条件の切り替え等も可能です。

### 3. 機械学習による分類
Tensowflowによる分類が実行されます。赤線と青線の交点が機械学習が予測した状態変化温度(相転移温度)です。
